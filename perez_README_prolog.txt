Prolog Student Courses.

To load the code, in prolog enter
[perez].

Students Turing, Galileo, Lovelace, Newton.
Departments Computer Science, Math, Physics

To find electives done for all students query
electivesDone(Name, Class).

To find students done with all requirements
done(Student).

To check if a student is done with core requirements query
doneReqCore(turing).

to check if a student is done with elective requirements.
doneReqElectives(newton).
